Did you think of something Jane could add to the bounce house?
Maybe a slight improvement in the code?
You have the power to make it so!

Suggestions follow the format of `[title] description`

```
[Let juniper spin!] Juniper should be able to do a spin/dance
```

The above example is from [an actual suggestion](https://github.com/just-jane-inc/just__stream/issues/250).

Any suggestion will produce a new issue in Jane's [list of GitHub issues](https://github.com/just-jane-inc/just__stream/issues).

Before creating a suggestion, please look for any existing issue that already describes your suggestion.
That would be super helpful.
